# Software Detailed Design - Math / 软件详细设计 - 数学层

## 中文

- `float32` 后端覆盖全库
- `Q31` 后端当前覆盖数学与部分控制算法，并在顶层 API 中支持 `PMSM FOC + Hall/Encoder + 2/3shunt`

## English

- `float32` backend covers the full library
- `Q31` backend currently covers math and selected control algorithms, with top-level API support limited to `PMSM FOC + Hall/Encoder + 2/3shunt`
